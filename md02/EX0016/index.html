Representação por códigos Hexadecimais

Abaixo um exemplo do titulo acima usando o style para colorir a fonte e o background usando os codigos Hexadecimais das cores correspondentes.

< style="background-color: #0000ff; color: #ffffff;"> Exemplo de cores </style>

Onde rr (vermelho), gg (verde) e bb (azul) são valores hexadecimais entre 00 e ff (igual ao decimal 0-255). Por exemplo, # ff0000 é exibido em vermelho, porque o vermelho é definido com seu valor mais alto (ff) e os outros dois (verde e azul) são definidos como 00. Outro exemplo, # 00ff00 é exibido como verde, porque o verde é definido com seu valor mais alto (ff) e os outros dois (vermelho e azul) são definidos como 00. Para exibir o preto, defina todos os parâmetros de cor para 00, como este: # 000000. Para exibir o branco, defina todos os parâmetros de cor para ff, assim: #ffffff.

Representação de cores por R-G-B

Abaixo um exemplo do titulo acima usando o style para colorir a fonte e o background usando os valores de RGB (RED, GREEN, BLUE ) das cores correspondentes.

< style="background-color: rgb;(255, 0 ,255) color: rgb(0, 0 , 255);"> Exemplo de cores </style>

Cada parâmetro (vermelho, verde e azul) define a intensidade da cor com um valor entre 0 e 255. Isso significa que existem 256 x 256 x 256 = 16777216 cores possíveis! Por exemplo, rgb (255, 0, 0) é exibido como vermelho, porque vermelho está definido com seu valor mais alto (255) e os outros dois (verde e azul) são definidos como 0. Outro exemplo, rgb (0, 255, 0) é exibido como verde, porque verde está definido com seu valor mais alto (255) e os outros dois (vermelho e azul) são definidos como 0. Para exibir o preto, defina todos os parâmetros de cor para 0, como este: rgb (0, 0, 0). Para exibir o branco, defina todos os parâmetros de cor para 255, como este: rgb (255, 255, 255).

Representação por HSL ( HUE, SATURATION, LUMINOSITY)

Abaixo um exemplo do titulo acima usando o style para colorir a fonte e o background usando valos HSL (HEU, SATURATION E LUMINOSITY) das cores correspondentes.

< style="background-color: hsl(240, 100%, 50%); color: hsl(0, 0%, 100%);"> Exemplo de cores </style>

Matiz é um grau no disco de cores de 0 a 360. 0 é vermelho, 120 é verde e 240 é azul. A saturação é um valor percentual, 0% significa um tom de cinza e 100% é a cor total. A claridade também é um valor percentual, 0% é preto e 100% é branco.

Representação de cores HSLA VISUAL STUDIO PALETA DE CORES

Abaixo um exemplo do titulo acima usando o style para colorir a fonte e o background usando valos HSLA (HEU, SATURATION, LUMINOSITY) e ALFA das cores correspondentes.

< style="background-color: hsla (240, 100%, 50%, 0.089); color: hsl(0, 0%, 100%, 0.26);"> Exemplo de cores </style>

Os valores de cor HSLA são uma extensão dos valores de cor HSL com um canal Alfa - que especifica a opacidade de uma cor.
